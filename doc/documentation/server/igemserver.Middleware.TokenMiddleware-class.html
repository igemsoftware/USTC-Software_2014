Class TokenMiddleware

        TokenMiddleware

a middleware that authenticate the client

if "Authorization" is found in HttpRequest header, the user matching this token will be logged in; else if "Authorization" is not provided, request.user will be an instance of AnonymousUser; else the request will be reject with return some error information.
